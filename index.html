In the Google homepage project, we were tasked with using JavaScript to make an alert pop up when the user clicked the search button. This alert contained the text that displayed what the user put in the text box. What if we wanted a google search to occur based on what the user put in the text box? What would that javascript look like?

The alert

alert("You searched for: " + userinput);

The Window Object: Global object that represents the browser window or a tab. It can be used with methods to interact with the browsers window.

Option 1: "window.location.replace(url);"

"Url" is where you would put your target site's url
Reloads current url with target url and user wont be able to click the top back button to get back to the previous page

function SameTab() { window.location.replace( "https://www.google.com/search?q="); }

Option 2: "window.location.href = url"

"Url" is where you would put your target site's url
Reloads current url with target url and user can click the top back button to get to previous page

function SameTab() { window.location.href = "https://www.google.com/search?q="; }

Option 3: "window.open(url,"_self")"

"Url" is where you would put your target site's url
"_self" parameter that specifies that that the place for the target url is the current url The default parameter is "_blank" is tell it to open a new tab for the target url so if the self paramter was removed the method would automatically open a new tab unlike options 1 and 2.

function SameTab() { window.open( "https://www.google.com/search?q=", "_self" ); }
